## Description

Simple games with numbers. Implemented in PHP using the command line.

## Requirements

PHP 7.4+

Composer 2.0+

## Setup

```sh
$ git clone https://github.com/softslot/php-project-lvl1.git

$ make install
```

## Brain Even

Answer "yes" if the number is even, otherwise answer "no".

```sh
$ make brain-even
```
